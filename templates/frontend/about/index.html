AXUM中文网(axum.rs)为你提供了使用axum进行企业级Web开发中所需要的大部分知识。从基础知识到企业级项目的开发，都有完整的系列教程。更难得的是，除了文字教程，我们还录制了配套的视频教程，方便你以多种形式进行学习。

技术栈

本站使用了大量的开源及免费项目，对这些伟大的项目表示感谢。

  • Rust
  • AXUM
  • PostgreSQL
  • Redis
  • Bootstrap
  • AdminLTE
  • InventIcons

更多 Rust 依赖，请参见 Cargo.toml 文件
